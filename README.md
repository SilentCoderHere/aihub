## ✨ Features

🎨 **Modern Material You design** with dynamic colors and auto dark/light mode  
🔄 **Fully customizable interface** - reorder, hide, or disable any AI assistant  
📱 **Smooth tabbed browsing** with persistent sessions and background support  
🔒 **Privacy-focused** - blocks trackers and unnecessary connections by default  
📏 **Adjustable font size** for comfortable reading  
⚡ **Lightweight and optimized** for fast performance  
💾 **Persistent conversations** that survive app restarts  
🎯 **Clean, intuitive interface** with no distractions
